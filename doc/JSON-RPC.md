> 参数：
 1. 等待时间(ms)

#### 返回值：私钥

#### 示例代码
##### 请求：

```shell
curl http://localhost:15645 -X POST --data '{"jsonrpc":"2.0","method":"consensus_changeWaitTime","params":[100000], "id": 3}' -H "Content-Type:application/json"
```

##### 响应：

```json
{"jsonrpc":"2.0","id":3,"result":null}
````
